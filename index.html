What is CellularUI, and Why is it neat?

CellularUI is a lightweight (4kb) library of jquery plugins providing a variety of common user interface controls that are easy to customize.

  • jMmenu
  • jAccordion
  • jTabs
  • jBlocklink
  • jScrolli

The goal is to take the same standard markup in the form of an HTML list and have the script apply the classes used (i.e. ".title", ".active", etc) to style and control the content.

Display Content as Tabs, Accordion, or Content-rotator with Views

  1. Create a View of Fields to collect the information you want to display
  2. Format: HTML List
  3. Settings: List class: jAccordion || jTabs || jScrolli
  4. Style the interface: /sass/style/style-ui.css

Mobile Menu

jMmenu({
"breakpoint": 650 // window size limit where mobile menu is displayed
})

Tabs

jTabs({
"active": 0, // the default element to display
})

Accordion

jAccordion({
"active": 0, // the default element to display
"duration": 500, // time of transition
"easing": "swing", // easing method
"single": false // display one element at a time
})

Blocklink

jBlocklink()

Scrolli

jScrolli({
"active": 0, // Index value of initial content to display
"speed": 500, // Duration of cycle
"pause": 3000 // Time to pause between cycles
})
