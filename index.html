jqMenu

Responsive menu that collapses to a button when the window size is less than the media query breakpoint.

<ul class="menu">
<li><a href="#">Item 1</a></li>
<li><a href="#">Item 2</a></li>
<li><a href="#">Item 3</a></li>
</ul>

<script>
$('.menu').jqMenu({
	"breakpoint": 650
});
</script>

jqFormal

Clear default values on focus, reinstate values if unchanged.

<form> 
<input type="text" value="Test"/>       
<input type="text" value="Required" class="required"/>              
<input type="submit" value="submit" />   
</form>
            
<script>
$('form').jqFormal();
</script>

jqTabs

Simple tabbed interface that takes a list, converts the first child node to the tab's title and the second child node to the tab's content.

<ul class="tabs">
<li>
    <p>Title 1</p>
    <div>
    <p>Content1 Content1 Content1 Content1 Content1</p>
    </div>
</li>
<li>
	<p>Title 2</p>
    <div>
    <p>Content2 Content2 Content2 Content2 Content2</p>
    </div>
</li>
</ul>

<script>
$('.tabs').jqTabs({
	"active": 0 // Index value of initial content to display
});
</script>

jqAccordion

Simple accordion takes a list, converts the first child node to the tab's title and the second child node to the tab's content.

    </div>
</li>
<li>
	<p>Title 2</p>
    <div>
    <p>Content2 Content2 Content2 Content2 Content2</p>
    </div>
</li>
</ul>

<script>
$('.accordion').jqAccordion({
"active": 0,// Index value of initial content to display

</script>

jqScrolli

Cycle through lists of content.

<p>This text is static, but this text <span class="scrolli">
<span>is scrolli</span>
<span>will rotate</span>
<span>cycles</span>
</span>.</p>

<script>
$('.scrolli').jqScrolli({
 "active": 0,// Index value of initial content to display
 "speed": 500,// Duration of cycle animation
 "pause": 3000// Time to pause between cycles
});
</script>
